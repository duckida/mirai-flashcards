# AI Flashcard Quizzer - Frontend

React Native Web application with Tamagui UI components.

## Setup

```bash
npm install
```

## Development

```bash
npm start
```

The app will be available at http://localhost:3001

## Build

```bash
npm run build
```

## Tech Stack

- React Native Web
- Tamagui (UI components and styling)
- Webpack (bundler)
- Babel (transpiler)
